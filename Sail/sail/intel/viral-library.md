## Patterns That Perform (extracted rules for Go)

### Hooks That Work
1. **Superlative + discovery:** "أكبر / أول / أقوى" + "فيه أحد سوى..."
2. **Revenue/number anchor:** "$7M ARR" / "75K views" / specific metrics
3. **National pride:** "بنت سعودية" / "السعوديين مبدعين"
4. **Contrarian:** "We killed our outbound engine" / doing the opposite of expected
5. **Free resource:** "مجاني 100% ومفتوح المصدر"

### Formats That Work
1. UGC with real demo (phone screen recording)
2. Bento grid brand viz (Nano Banana Pro)
3. Thread/article with specific numbers and phases
4. Tool discovery posts ("I found this...")
5. Before/after comparisons
