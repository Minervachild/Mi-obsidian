# Ad References — M's Picks

## Key Patterns from References:

### 1. OneShot (Saudi hydration brand)
- **Pattern:** Product lineup front & center + occasion hook (National Day) + bold discount + clean CTA
- **What works:** Products ARE the hero. Arabic bold headline. Simple "Order now" button. Dark background makes products pop.

### 2. Rite Plant Protein
- **Pattern:** ⭐⭐⭐⭐⭐ social proof + emotional headline ("أفضل بروتين بدون تأنيب ضمير") + product hero + 3 benefit badges at bottom
- **What works:** Stars grab attention. Headline speaks to FEELING not features. Benefit badges (builds muscle / recovery / delicious) are scannable. Monochrome background matches product.

### 3. MyProtein ME
- **Pattern:** Urgency tag ("عرضك الحصري") + MASSIVE discount number + product + benefit callouts as floating badges + coupon code circle + Shop now CTA
- **What works:** The discount number is HUGE — impossible to miss. Floating badges feel dynamic. Coupon code creates exclusivity. Arabic + English mix.

### 4. BEIS Luggage
- **Pattern:** "I LOVE BACK PAIN" — No one, ever. + Product + one-liner benefit + CTA
- **What works:** SARCASM/IRONY as hook — instantly relatable. Quote format makes it feel like a real person talking. Minimal copy. Product speaks for itself.

## Takeaways for Sail:
1. **Bold headline that hits emotion** — not features
2. **Product/mockup as hero** — show the WhatsApp conversation, not abstract graphics
3. **Scannable benefit badges** — 3 max at bottom
4. **Clean CTA** — one button, one action
5. **Sarcasm/irony works** — "أحب أدخل فواتير" — ما أحد قالها بحياته
6. **Stars/social proof** if available
7. **Dark backgrounds** make content pop
8. **Urgency tags** — "عرض محدود" / "هالأسبوع بس"
