# 📊 وكيل التحليل التسويقي — Marketing Analysis Agent

## المشكلة
- صاحب المتجر يصرف إعلانات بدون ما يعرف وش يشتغل ووش لا
- البيانات موجودة بس ما أحد يقرأها
- قرارات التسويق بالإحساس مو بالأرقام
- الـ bounce rate عالي وما تدري ليش
- تصرف 5000 ر.س إعلانات والمبيعات ما تزيد

## الحل — 3 وكلاء يشتغلون مع بعض

### 1. وكيل التحليل (Analyst)
- يقرأ بيانات متجرك 24/7 (زد، شوبيفاي، أي منصة)
- يتتبع: CR، bounce rate، AOV، مصادر الترافك، سلوك العميل
- يكتشف المشاكل قبل ما تحس فيها
- ينبهك فوري لو فيه تغير غريب

### 2. وكيل التوصيات (Recommender)
- يحلل سلوك كل عميل
- يقترح منتجات مخصصة لكل زائر
- يحدد أفضل وقت للتواصل
- يقترح عروض وبوكسات بناءً على أنماط الشراء

### 3. وكيل التقارير التنفيذية (Executive Brief)
- تقرير أسبوعي للـ CEO/صاحب البزنس
- ملخص: وش صار، وش المشكلة، وش الحل
- بالعربي، واضح، بدون مصطلحات معقدة
- يوصلك واتساب أو إيميل

## لمين
- أصحاب المتاجر الإلكترونية (زد، سلة، شوبيفاي)
- شركات الـ e-commerce (10K+ زيارة/شهر)
- أي أحد يصرف إعلانات ويبغا يعرف العائد الحقيقي

## الأرقام (مثال حقيقي — السروي)
- Bounce rate: 99% → 50% (بعد ما فهمنا البيانات)
- CR: 0.3% → 1.3% (4x تحسن)
- أكبر تسريب محدد: صفحة المنتج → السلة (48% يشوف، 8% يضيف)
- الحل: social proof + bestseller section + وصف أقوى
