## Color Palette
- **Primary text:** #FFFFFF (white)
- **Shadow/outline:** #000000 40% opacity
- **Accent (arrows/CTA):** #00B4D8 (Sail blue)
- **Background overlays:** Dark gradient 0-40% opacity

---

## Font Recommendations
- **Arabic:** Tajawal Bold / Dubai Bold
- **Numbers/English:** Poppins SemiBold

---

*All text should be legible on 9:16 mobile screens — minimum 24pt font size*
